App download:

- unzip the main file from moodle
- run main.exe

Usage instructions:

If only csv files as input files available: file -> convert -> select the file to be converted and wait for the program to finish
If bin files available: file -> open -> select bin file and wait for program

Interface overview:

Left panel:
    - file name/size
    - data points
    - signal duration
    - num of tumor peaks on both signals
    - toggles for signals, peak types and baseline

Main plot area:
    - Plot with each toggled visual
    - toolbar for plot interactions
    - range slider (to use it, drag the bar from both sides to fit to a range you want and then drag the bar itself to move along the plot with the specified range)
